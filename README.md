## ✨ Features

- **Hero animato** — le parole entrano in sequenza con animazioni CSS pure e un cursore lampeggiante
- **Header sticky** con effetto glassmorphism
- **Menù hamburger** con animazione fluida, senza JavaScript
- **Barra scorrevole** che si mette in pausa al passaggio del mouse
- **Foto fluttuante** nella sezione Chi sono
- **Footer** con micro-interazioni in hover
- **Layout responsive** con Bootstrap e media query personalizzate
- **SCSS modulare** con variabili per colori, font e ombre
